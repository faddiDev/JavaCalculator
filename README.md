# JavaCalculator

Java Calculator

<b>Installation:</b>
<ul>
  <li>Make sure java is installed on your machine</li>
  <li>Open cmd.exe</li>
  <li>Target your folder application</li>
  <li>Compile first: javac Calculator.java</li>
  <li>Run it: java Calculator</li>
</ul>
